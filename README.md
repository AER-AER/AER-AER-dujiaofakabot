# TG独角发卡bot
  简介：
  TG机器人数据与网站数据同步
  
  购买商品功能与独角发卡数据同步，即是独角发卡机器人
  
  此前该机器人的功能具有目前最实用的功能
  
  支持okpay支付功能
  
  后台：简单快捷、可视化
  
  管理后台模式：机器人端、web端
## 独角发卡分站式管理
   意思是发卡网站可以分站

## 个别功能解释
会员功能：24小时自动开VIP，全自动付费订阅管理。支持自助入群、会员到期自动提醒/移除、多级权限控制。

发卡功能：全自动数字商品售卖系统。支持多种数字货币支付、单个与批量的文件上传，具备完善的卡密库存管理、多级商户后台、自动发货与订单查询功能。
